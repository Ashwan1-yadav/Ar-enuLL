# Ar$enuLL Backend

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install the dependencies
4. Create a .env file and add the following variables
   - PORT
   - MONGO_URL
5. Run the server

```bash
npm install
```

```bash
npm start
```
